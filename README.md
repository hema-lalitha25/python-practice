# Python GitHub Samples

This repository contains small, beginner-friendly Python programs you can upload to GitHub.

Included files:
- hello.py : simple input and print example
- cli_calc.py : tiny calculator (demo only)
- prime_checker.py : primality test
- file_utils.py : read/write text and CSV
- simple_server.py : tiny HTTP file server

Usage:
1. Clone the repo or download the files.
2. Run `python hello.py` etc.
3. See upload instructions in UPLOAD_INSTRUCTIONS.txt
